# HomyHive

HomyHive is a web application that allows users to find and book rental properties. It is built with Node.js, Express, and MongoDB.

## Features

- User authentication (signup, login, logout)
- Browse and search for listings
- View listing details
- Book listings
- Host registration and property creation
- Admin dashboard for managing host requests

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository
2. Install the dependencies
   ```
   npm install
   ```
3. Start the application
   ```
   npm start
   ```
